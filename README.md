# ExMED SYSTEM  
A DIGITAL SOLUTION FOR EFFICIENT HEALTHY RECORD MANAGEMENT IN UGANDA. 
The ExMed system solves the problem of mismanagement of healthy records since most healthy facilities in Uganda rely on manual based systems to store healthy records. Manual paper based sytems create several serious like slow record retrieval,high data loss and errors, Administrative burden as well as many problems in most healthy facilities. 

We developed A multilingual,scalable,user friendly interface,secure data management system to digitise medical records, enhance data accessibility and improve the efficiency and quality of health care service in uganda. 

 INSTRUCTIONS ON HOW TO RUN ExMED SYSTEM 
   ExMED System

Project Description
ExMED System is a web-based medical management system developed using PHP, HTML, CSS, JavaScript, and MySQL. The system is designed to improve the management of medical records and related healthcare services through a simple and user-friendly web interface.

Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL

 Setup Instructions

1. Clone or download this repository from GitHub.
2. Copy the project folder into the `htdocs` directory of XAMPP.
3. Start Apache and MySQL using the XAMPP Control Panel.
4. Import the provided SQL database into phpMyAdmin.
5. Open your web browser and navigate to:

   http://localhost/ExMED/

6. The system will be ready for use.
